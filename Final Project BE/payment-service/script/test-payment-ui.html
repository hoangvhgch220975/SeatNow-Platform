Tạo thanh toán đặt cọc

Gọi endpoint tạo QR hoặc link thanh toán cho booking. Kết quả sẽ tự lưu lại transaction ID và reference code.

Xem transaction

Tra cứu transaction theo ID sau khi tạo giao dịch hoặc sau khi webhook xử lý xong.

Giả lập webhook

Điền payload webhook rồi gửi trực tiếp tới payment-service. Nếu đang verify chữ ký, bạn phải cung cấp signature/hash hợp lệ.

Luu y: webhook se loi neu signature/hash khong hop le.

Mở return URL

Dùng để test luồng redirect sau thanh toán. Nút này sẽ mở endpoint return trong tab mới.

Kết quả trả về

Mọi response, error và URL đang gọi sẽ hiển thị ở đây để bạn theo dõi nhanh.
